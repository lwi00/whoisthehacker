🛡️ WHO'S THE HACKER : Manuel de l'Utilisateur
Minimum de joueurs : 7
🎯 BUT DU JEU
Pour les Employés : Éliminer tous les Hackers infiltrés.

Pour les Hackers : Éliminer tous les Employés du réseau jusqu'à être majoritaires.

📇 LES CARTES DU RÉSEAU
🔴 Les Menaces

Le Hacker (Loup-Garou) : Chaque nuit, il infiltre et compromet un compte utilisateur. Le jour, il se fait passer pour un Employé pour ne pas être banni. À noter que le Hacker ne peut pas de hacker lui même pendant la nuit.

Le Hacker Infiltré (Voleur) : Durant la première nuit, il peut échanger son identité avec l'une des cartes non distribuées. S'il voit deux cartes "Hacker", il est obligé d'en prendre une.

🔵 Le Personnel de Sécurité

L'Analyste de Données (Voyante) : Chaque nuit, elle scanne un joueur pour connaître sa véritable identité réseau.

L'Ingénieure Système (Sorcière) : Elle possède deux scripts uniques : un Correctif pour sauver la victime des Hackers, et un Contre-virus pour éliminer un suspect la nuit. Elle ne peut utiliser chaque script qu'une seule fois par partie.
Attention : Cela ne fonctionne que pour un utilisateur piraté pendant la nuit.

L'Agent de Cybersécurité (Chasseur) : S'il est éliminé (par les hackers ou par vote), il a le pouvoir de répliquer en neutralisant immédiatement un autre joueur de son choix.

🟢 Les Utilisateurs & Support

L'Employé (Simple Villageois) : Il n'a aucun privilège d'accès particulier. Il doit utiliser son intuition et l'analyse des logs (débats) pour voter.

La Stagiaire IT (Petite Fille) : Elle peut espionner les Hackers pendant leur phase d'attaque (en entrouvrant les yeux). Si elle est repérée, elle est éliminée immédiatement à la place de la victime prévue.

L'Administrateur Réseau (Cupidon) : La première nuit, il crée une connexion sécurisée entre deux utilisateurs. Si l'un des deux est déconnecté, l'autre subit un plantage système immédiat et est éliminé aussi.

Le Responsable (Capitaine) : Élu par le groupe, son vote compte pour deux voix. S'il est éliminé, il désigne son successeur dans son dernier message.

🌑 PHASE DE MAINTENANCE (LA NUIT)
L'Admin (Maître du jeu) demande à tout le monde de fermer les yeux.

Hacker Infiltré (Voleur) : (Tour 1) Il choisit son rôle parmi les cartes au centre.

Administrateur Réseau (Cupidon) : (Tour 1) Il lie deux joueurs.

Utilisateurs Liés (Amoureux) : (Tour 1) Ils se reconnaissent.

Analyste de Données (Voyante) : Elle sonde l'identité d'un joueur.

Les Hackers : Ils se concertent et désignent une victime. La Stagiaire IT peut tenter d'observer.

Ingénieure Système (Sorcière) : Elle décide d'utiliser, ou non, ses capacités de Correctif ou de Contre-virus.

☀️ ANALYSE DES LOGS (LE JOUR)
Rapport d'Incident : L'Admin annonce qui a été éliminé durant la nuit. Le joueur révèle son rôle et ne peut plus communiquer.

Le Débat : Les survivants discutent pour identifier les Hackers.

Le Vote : Le joueur désigné par la majorité est banni du réseau (éliminé) et révèle son identité. En cas d'égalité, le Responsable tranche.

Mise en veille : Le réseau s'endort et un nouveau cycle commence.
